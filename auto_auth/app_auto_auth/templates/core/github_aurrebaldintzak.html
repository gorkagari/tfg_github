GitHuben saio-hasieraketa inplementatzeko aurrebaldintzak


  • Jo GitHubera eta hasi saioa zure GitHub kontuarekin (konturik ez badaukazu sor ezazu bat).

  • Behin saioa hasita, egin klik zure perfil irudian eta joan "Developer settings" atalera. Bertan aukeratu "OAuth Apps" sekzioa eta egin klik "New OAuth App" botoian".

  • Formulario bat aurkeztuko zaigu, non nahitaez hurrengo atalak bete beharko ditugun:


    • “Application name”: Gure aplikazioari eman nahi diogun izena.

    • “Homepage URL”:Web aplikazioaren URL helbidea.

    • “Authorization callback URL”: Idatzi http://zuredomeinua:8000/accounts/github/login/callback/ esteka.


  • Orain proiektua sortuta daukagula, bere "Client ID" eta "Client Secret" ditugu eskuragarri.

  • Bukatzeko, zure proiektuaren SETTINGS.PY fitxategian, INSTALLED_APPS zerrendan, gehitu 'allauth.socialaccount.providers.github', eta exekutatu py manage.py migrate komando lerroan.
