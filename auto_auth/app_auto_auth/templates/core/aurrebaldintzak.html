Instalatu django-allauth liburutegia (Nahitaezkoa)


- Exekutatu pip install django-allauth komandoa komando-lerroan.

'settings.py' fitxategian:

  • INSTALLED_APPS atalean, gehitu hurrengo lerroak:

    'django.contrib.sites',
    'allauth',
    'allauth.account',
    'allauth.socialaccount',

  • TEMPLATES atalean, context_processors zerrendan, ez balego gehitu lerro hau:

    'django.template.context_processors.request',

  • AUTHENTICATION_BACKENDS atala gehitu, hurrengo edukiarekin:

    AUTHENTICATION_BACKENDS = (
    'django.contrib.auth.backends.ModelBackend',
    'allauth.account.auth_backends.AuthenticationBackendâ€™,
    )

  • SITE_ID aldagaia gehituko:

    SITE_ID = 1

'urls.py' fitxategian:

  • urlspatterns zerrendan gehitu:

    path('accounts/', include('allauth.urls')),
